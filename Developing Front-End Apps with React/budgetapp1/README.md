🌿 Paradise Nursery – Shopping Cart Application
📌 Project Overview

Paradise Nursery is a dynamic and interactive plant shopping web application built using React and Redux. This project allows users to explore a variety of houseplants, view detailed product information, and manage their shopping cart efficiently.

The application simulates a real-world e-commerce experience with features like product browsing, cart management, and dynamic price calculation.

🚀 Features
🌱 Landing Page
Attractive background image
Company name and branding
Brief description about Paradise Nursery
“Get Started” button to navigate to the product listing page
🪴 Product Listing Page
Displays multiple houseplants with:
Image (thumbnail)
Name
Price
Plants are grouped into different categories
“Add to Cart” functionality:
Adds item to cart
Disables button after adding
Updates cart count dynamically
🛒 Shopping Cart
Displays all added items with:
Image, name, and unit price
Features:
Increase / Decrease quantity
Remove items from cart
Total items count
Total price calculation
Buttons:
Checkout (Coming Soon)
Continue Shopping
🔝 Navigation Bar
Available on all pages
Links to:
Home
Plants
Cart
Dynamic cart icon showing total items
